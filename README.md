# تطبيق الفنادق الجزائرية - Algerien Hotel

تطبيق لحجز الفنادق في الجزائر مبني لمنصة Pi Network.

## المميزات
- عرض قائمة الفنادق في مختلف المدن الجزائرية
- تفاصيل كاملة عن كل فندق والغرف المتاحة
- نظام حجز مع تأكيد
- ربط محفظة Pi Network (قيد التطوير)
- تصميم متجاوب مع جميع الأجهزة

## طريقة التشغيل
1. افتح المشروع في محرر النصوص (مثل VS Code)
2. شغل المشروع باستخدام Live Server أو أي خادم محلي
3. افتح المتصفح على العنوان http://localhost:5500

## للتشغيل على Pi Browser
1. ارفع الملفات على استضافة (مثل Netlify أو Vercel)
2. افتح الرابط داخل Pi Browser

## التقنيات المستخدمة
- HTML5
- CSS3
- JavaScript (ES6+)
- Pi Network SDK
